# SARC-Packer v0.2
A tool for packing folders into SARC files.  

## Usage:
 * <code>main [Option...] folder</code>  
All the options are optional.
 
## Options:
 * -o (Output file)
 * -bom (Endianness)
 * -padd (Padding value)  
  
("padd" here is a short for "padding", not a typo of "pad")
